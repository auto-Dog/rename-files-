# rename-files-
对当前文件夹下同类型文件批量格式化命名


1.脚本需要移动到相应文件夹内运行


2.脚本可以将同一类型文件按照格式全体命名、修改（需要自行更改）


3.更改名称前，会将本文件夹下（一级目录）所有文件名和后缀输出到oriName备份
********************************************************************************************
4._使用方法_：
	先输入需要更改的一堆文件的后缀名（如mmm.spl 对应输入spl） （如果找不到该类型的文件会在cmd中显示未找到，这时候退出就行了）；
	然后按指引输入命名格式 比如要统一命名成 示例-abcd-数字编号.spl，需要输入 示例-abcd-
	然后就可以了
